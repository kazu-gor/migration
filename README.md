# migration

新しいMacに移行する際に使用する

下記コマンド入力
```zsh
make all
```

makeコマンドでできる設定
- Homebrew
- vim-plug
- dotfiles
- symbolic link

# その他設定

## キーボード設定
### [システム環境設定]→[キーボード]→[修飾キー]

Caps lock → Controlに変更

### [システム環境設定]→[キーボード]
キーのリピート→Max速い

リピート入力認識までの時間→Max短い

### [システム環境設定]→[キーボード]→[入力ソース]

ライブ変換オフ


### [システム環境設定]→[トラックパッド]

トラックパッドの速度変更


### [システム環境設定]→[ディスプレイ]→[サイズ調整]

デフォルト→スペースを拡大

### [システム環境設定]→[キーボード]→[ショートカット]→[Spotlight]

Spotlight検索を表示のチェックを外す

### [システム環境設定]→[サウンド]→[サウンドエフェクト]

好きなやつに変更
